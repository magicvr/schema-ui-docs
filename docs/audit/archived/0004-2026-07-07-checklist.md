## 🔴 N 组：Schema 与文档同步遗漏（建议 v0.2.2）

### N1. `component-registry.json` 中 `tabs.props.items.content` 未引用 Node Schema

- [x] **修复方式：** 将 `"type": "object"` 替换为 `"$ref": "node.schema.json"`
- [x] **涉及文件：** `docs/schemas/component-registry.json`
- [x] **参考：** `action.schema.json` 中 `modal.content` 的 `$ref` 写法
- [x] **验证方法：** 确认校验工具能通过 `$ref` 展开检查 `tabs` 的 `items[].content` 中的 Node 结构合法性

---

### N2. `component-registry.json` 中 `table.props.columns.items` 缺少 `visibleWhen`/`reactions`/`permissions`/`labelKey` 字段

- [x] **修复方式：** 在 `columns.items.properties` 中补充：
  - `labelKey`: `{ "type": "string" }`
  - `visibleWhen`: `{ "$ref": "node.schema.json#/definitions/VisibleWhen" }`
  - `reactions`: `{ "type": "array", "items": { "$ref": "reaction.schema.json" } }`
  - `permissions`: `{ "$ref": "node.schema.json#/definitions/Permissions" }`
- [x] **涉及文件：** `docs/schemas/component-registry.json`
- [x] **验证方法：** 确认 L2 校验能检测到 `columns` 中 `visibleWhen`/`reactions`/`permissions` 的格式错误

---

### N3. `component-registry.json` 中 `table.props.actions.items` 缺少字段和约束

- [x] **修复方式：**
  1. 补充 `visibleWhen`/`reactions`/`permissions`/`labelKey` 字段定义（同 N2 方式）
  2. 补充 `required: ["key"]`
  3. 补充 `additionalProperties: false`
- [x] **涉及文件：** `docs/schemas/component-registry.json`
- [x] **验证方法：** 确认 L2 校验能检测到 `actions` 中的非法字段和缺失必填字段
