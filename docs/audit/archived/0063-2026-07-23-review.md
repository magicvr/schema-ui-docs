### V273（P1）formRecord 缺失路径 — 已关闭

**裁决：** conformance 可观测值为 JSON `null`；语义等价空初始值，不中止整次回填。已写入 ADR-0021 D5a、`03` form.recordSource 摘要；双端 reference 注释对齐。

### V274（P1）/ V281（P2）batch selection 不变量 — 已关闭

**裁决：** `buildBatchRequest` 入口调用与 table-query-state 同一 `normalizeSelection`（过滤非标量、去重保序、重算 count）；规范化后为空 → `EMPTY_SELECTION`。fixtures：`batch-request-count-keys-mismatch-normalized`、`batch-request-non-scalar-and-dup-keys-normalized`、`batch-request-all-non-scalar-keys-empty`。同步 ADR-0022 D3/D5d、`07` §3.5。

### V275（P1）2.2 版本策略 — 已关闭

**裁决：** 写死于 ADR-0022 状态段 + [`13-v2.2-release-goals.md`](../../13-v2.2-release-goals.md) + [`migrations/2.1-to-2.2.md`](../../migrations/2.1-to-2.2.md)。发布前可用 `2.1` + capability；`2.2.0` tag 时样例/相关 fixtures 升 `"2.2"`；发布后使用 2.2 字段的页面必须 `"2.2"`。

### V276（P2）文档过期 — 已关闭

ADR-0020/0021 示例 `protocolVersion: "2.1"`；`admin-list-edit-lifecycle` 删除过期「2.1 发布前可声明 2.0」；`11` §7 0020/0021 标为 accepted。

### V277（P2）Phase C 交付 — 已关闭

迁移短文、扩展示例 `admin-list-batch`、发布目标文档已齐；**`2.2.0` 制品打包**明确为发布 tag 动作（`13` G1/G4 未勾选项），不阻塞语义闭合。

### V278（P2）L2 method 诊断 — 已关闭

缺失 → 文案含 `MISSING_RECORD_SOURCE_METHOD`；非法 → `RECORD_SOURCE_METHOD_NOT_GET`。样例：`_samples/audit-0063-record-source-missing-method-bad.yaml`。

### V279（P2）L3a route 语义 — 已关闭

`route` 纳入 `KNOWN_CONTEXT_ROOTS`；expression 挂载点 `FORBIDDEN_CONTEXT_NAMESPACE`。样例：`_samples/audit-0063-context-route-in-visible-when-bad.yaml`。同步 `02` §11.3。

### V280（P2）modal route — 已关闭

ADR-0021 D7：modal 内 recordSource route 注入 **非** 2.1/2.2 互操作门禁；若用 route 绑定应提供与整页相同快照形状；MVP 推荐字面量 path/query。
