# 审计 0055 跟踪清单

## V218（🔴）严格版本协商

- [x] 新增 ADR-0009，精确匹配 `supportedVersions`，删除最近版本猜测和 `minCompatibleVersion`。
- [x] 标准入口拒绝缺失、畸形、未知 MAJOR 和未知 MINOR。
- [x] 定义显式 legacy adapter 边界和输出契约。
- [x] 增加版本与 capability 一致性 fixtures、JavaScript reference 和 14 个正反例向量。

## V219（🔴）query 线级序列化

- [x] 新增 ADR-0010，采用标量-only RC 方案。
- [x] 定义 `null` / `undefined` / 空字符串 / number / boolean 规则。
- [x] 定义 UTF-8、RFC 3986、顺序、重复 key、已有 query 和 fragment 规则。
- [x] Schema/L2 拒绝 query 对象、数组、空 key 和非有限数值。
- [x] 四类 query 来源复用同一算法；JavaScript 与 Python reference 通过同一 16 个字节级向量。

## V220（🔴）保留参数与冲突优先级

- [x] 新增 ADR-0011，将 `page` / `pageSize` / `sort` 定为保留参数。

- [x] 固定 URL query、静态 params、搜索字段、Renderer 状态的合并顺序。
- [x] 固定提交、清空、翻页、排序时的页码和筛选状态转换。
