# 审计 0055 跟踪清单

## V218（🔴）严格版本协商

- [ ] 新增 ADR-0009，精确匹配 `supportedVersions`，删除最近版本猜测。
- [ ] 标准入口拒绝缺失、畸形、未知 MAJOR 和未知 MINOR。
- [ ] 定义显式 legacy adapter 边界和输出契约。
- [ ] 增加版本与 capability 一致性 fixtures 和参考实现。

## V219（🔴）query 线级序列化

- [ ] 新增 ADR-0010，采用标量-only RC 方案。
- [ ] 定义 `null` / `undefined` / 空字符串 / number / boolean 规则。
- [ ] 定义 UTF-8、RFC 3986、顺序、重复 key、已有 query 和 fragment 规则。
- [ ] Schema/L2 拒绝 query 对象、数组和非有限数值。
- [ ] 四类 query 来源复用同一算法并通过字节级向量。

## V220（🔴）保留参数与冲突优先级

- [ ] 新增 ADR-0011，将 `page` / `pageSize` / `sort` 定为保留参数。

- [ ] 固定 URL query、静态 params、搜索字段、Renderer 状态的合并顺序。
- [ ] 固定提交、清空、翻页、排序时的页码和筛选状态转换。
