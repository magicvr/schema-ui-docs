## 🔴 M 组：ajv 实测发现的严重问题（建议立即修复，v0.2.3）

### M1. `action.schema.json` `additionalProperties: false` 导致所有合法 action 校验不通过

**ajv 实测结果：** `05-scenarios/form-with-reactions.md` 中的 `createOrder` 示例报 `additionalProperty: 'method'` 等 4 个错误。

- [x] **修复方式：** 将 `method`/`url`/`bodyMapping`/`onSuccess`/`onError`/`modalId`/`content`/`handler` 全部提到顶层 `properties` 中声明（各自标为可选），`allOf`/`if-then` 只负责按 `type` 做 `required` 约束。移除 `then.properties` 中重复的字段定义。
- [x] **涉及文件：** `docs/schemas/action.schema.json`
- [x] **验证方法：** 用 ajv 编译修改后的 Schema，然后校验以下三个示例：
  - `05-scenarios/form-with-reactions.md` 中的 `createOrder`（request 类型）
  - `05-scenarios/form-with-reactions.md` 中的 `confirmDelete`（modal 类型）
  - `docs/07-actions-contract.md §8` 中的完整示例（request + navigate + modal）
- [x] **回归检查：** 确认所有示例校验通过，无 `additionalProperty` 错误。
